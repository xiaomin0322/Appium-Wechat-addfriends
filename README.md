# Appium-Wechat-addfriends
通过appium 操作微信加好友，并判断状态（异常，查询不存在 等）

另外严重推荐 https://github.com/summerlove66/YeShen-Wechat-Automatic   同时操作多个微信自动化加入。这个也是我在公司内部跑过 比较稳定。

jdk
nodejs
安卓sdk
appium-desktop
配置手机usb开发模式，充电不锁屏。允许usb安装程序
